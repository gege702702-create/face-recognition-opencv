# Face Recognition using OpenCV

A simple project that uses the OpenCV library together with a pretrained Haar Cascade model to detect human faces — either in a static image or live through a webcam.

This project implements Task 1 from the task list: Face Recognition.

## ⚙️ Step-by-Step Explanation

### 1. Install the required library
The project only needs OpenCV:
pip install opencv-python

Or install it from the requirements file:
pip install -r requirements.txt

### 2. Load the Haar Cascade model
haarcascade_frontalface_default.xml is a pretrained model provided by OpenCV, specialized in detecting frontal human faces. It is loaded in the code with:
face_cascade = cv2.CascadeClassifier("haarcascade_frontalface_default.xml")

### 3. Convert the image to grayscale
Haar Cascade detection works faster and more accurately on grayscale images rather than color images:
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

### 4. Detect faces
Using the detectMultiScale function, the image is scanned at multiple scales to find every face present in it:
faces = face_cascade.detectMultiScale(
    gray,
    scaleFactor=1.1,   # how much the image size is reduced at each scan step
    minNeighbors=5,    # how many overlapping rectangles are required to confirm a real face
    minSize=(40, 40)   # smallest face size that can be detected
)

### 5. Draw a rectangle around each detected face
For every detected face, a green rectangle is drawn around it with the label "Face":
cv2.rectangle(img, (x, y), (x + w, y + h), (0, 255, 0), 2)

### 6. Display and save the result
The final image is shown in a window and also saved to images/Face_detected.jpg.

### 7. Live webcam mode
In addition to detecting faces in a static image, the project provides a second mode that runs on a live video stream from the webcam. The same detection steps (grayscale → detect → draw rectangle) are repeated on every frame in real time until the user presses q to quit.

## ▶️ How to Run
python face_recognition.py

You will be prompted with two options:
1. Detect faces in the sample image (images/Face.jpg)
2. Detect faces live using the webcam

- Choose 1 to detect faces in the attached sample image.
- Choose 2 to run live detection through the webcam (press q to exit the window).

---

## 🧠 How It Works (Concept)

The Haar Cascade algorithm is trained beforehand on thousands of positive images (containing faces) and negative images (without faces), learning the characteristic light/dark contrast patterns of facial features (eyes, nose, mouth). After training, these patterns are stored in an XML file, which is used directly here (no retraining needed) to search for the same patterns inside any new image.
